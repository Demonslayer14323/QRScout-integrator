How to use:

  1. Use QRScout to collect scouting data
  2. Copy the data from the QR model
  3. Paste the data into the box below
  4. Enter your Apps Script URL or Sheet ID (see note below)
  5. Click "Send to Google Sheets"

Note: Due to Google's OAuth verification requirements, we recommend using an Apps Script URL instead of direct Sheet ID access. Apps Script URLs work without Google Sign-In and are more reliable. See CONFIG.md for setup instructions.
